Cross-platform.org provides an overview of selected cross-platform mobile development technologies, defines criteria and offers a decision tool. This decision tool enables the user to enter requirements (criteria), evaluates the considered technologies and displays a ranking.


The website is structured as follows:

  • Technologies: Overview of considered technologies with detailed information, evaluated according to the defined criteria
  • Criteria: Overview of provided criteria
  • Decision Tool: A tool to match user requirements against selected criteria and to create a ranking of technologies
  • Profiling: Customization of technologies and criteria via JSON (beta version)
